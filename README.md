# 🚀 Featured Projects

## 🛡️ AURA — Advanced Unified Recon & Assessment

> A modular offensive security platform focused on reconnaissance, automation, vulnerability discovery, and security workflow optimization.

### ✨ Key Features

* Automated Reconnaissance Pipelines
* Endpoint & Asset Enumeration
* Security Workflow Automation
* Scalable Modular Architecture
* Security Reporting & Data Correlation
* Python + Django Powered Backend

### 🔥 Why It Matters

AURA is designed to reduce manual recon effort while providing structured and scalable security assessments. The project reflects real-world red team workflow automation and offensive security engineering.

---

## 📡 AirFence — Wireless Threat Detection & Defense Platform

> AirFence is a wireless security monitoring and attack detection platform built around ESP32 and IoT-based security research.

### ✨ Key Features

* Rogue Access Point Detection
* Evil Twin Detection
* Wireless Monitoring
* ESP32-based Edge Detection
* Offline IoT Logging Support
* Django-based Dashboard Integration
* Security Alerting & Monitoring

### 🔥 Why It Matters

AirFence combines IoT engineering with cybersecurity to create practical wireless defense tooling. It demonstrates embedded security development, network analysis, and real-time monitoring capabilities.
